# NumerLett Landing Page

This project replicates the `numerlett.html` landing page as a fully componentized Next.js + Tailwind UI for the NumerLett agency.

## Structure

- [app/page.tsx](app/page.tsx) composes the full page from section components.
- [components/sections](components/sections) contains section-by-section UI.
- [app/globals.css](app/globals.css) defines NumerLett design tokens and animations.

## Getting Started

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the landing page.
